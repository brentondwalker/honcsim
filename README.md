# topocodocov
Compute information coverage of geographically distributed network coded data using topological methods

 Quickstart
------------

1. Build the package
```
  ant
```

2. Run an example
```
  ./simrun topocodocov.<example-name> [args]
```
where example-name is one of the example classes:
* BubbleBox
* TrippleBubbleBox
* NonLifting
* Wormhole
* LayeredLoop
* LayeredSimplex
* CoverageGrid
